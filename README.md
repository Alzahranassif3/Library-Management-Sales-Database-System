
# Library Management & Sales System

## 📌 Overview

This project is a full-stack database system designed for managing a multi-branch library business.
It handles inventory, sales, suppliers, customers, and staff operations efficiently.

---

## 🚀 Technologies Used

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MySQL

---

## ⚙️ Features

* Inventory and stock management
* Sales and purchase tracking
* Supplier and customer management
* Multi-branch support
* Reporting and analytics (top-selling products, revenue, low stock)

---

## 🧠 Database Design

* Designed using ER modeling
* Contains 14+ relational tables
* Normalized up to **Third Normal Form (3NF)**
* Includes complex relationships (one-to-many, ternary)

---

## 💻 System Components

* Backend API using Flask
* Frontend interface using HTML/CSS/JS
* MySQL database for data storage

---

## 📊 Example Functionalities

* Track product availability across branches
* Generate sales reports
* Identify low-stock items
* Manage supplier purchases

---
